Fish Bot

Mentor @e-YAntra, IITB
  • Designed and implemented Bio-mimetic Robotic Fish which can swim underwater and can be wirelessly controlled via NRF24L01 module
  • Maneuvering of fish achieved with active control of Caudal(Tail) and Pectoral(Side) Fins
  • Waterproofing using gaskets and flexible tail made of silicon rubber
  • This project was formulated and the concept was validated in eYSIP '17 mentored by Rucmenya Bessariya, Lohit Penubaku and Fayyaz Pocker
    • Solidworks
    • 3D printing
    • Wireless control using NRF module
